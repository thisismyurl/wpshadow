# WPShadow Philosophy Checklist

Before committing ANY code, verify:

## 🌍 Accessibility First (CANON)
- [ ] Works with keyboard only (Tab, Enter, Space, Arrow keys)
- [ ] Screen reader can read all content (test with NVDA/VoiceOver)
- [ ] Zoom to 200% - still readable
- [ ] Color contrast meets WCAG AA (4.5:1 text, 3:1 UI)
- [ ] All images have alt text
- [ ] All buttons have clear labels
- [ ] No time limits without user control
- [ ] Respects prefers-reduced-motion
- [ ] Touch targets min 44x44px
- [ ] Focus indicator always visible

## 🎓 Learning Inclusive (CANON)
- [ ] Documentation exists in text format
- [ ] Video tutorial available (or planned)
- [ ] Real-world example included
- [ ] Plain English (no jargon without explanation)
- [ ] Progressive disclosure (simple first, details available)

## 🌐 Culturally Respectful (CANON)
- [ ] No idioms or colloquialisms
- [ ] Dates use locale formatting (not hardcoded)
- [ ] Numbers use locale formatting
- [ ] All strings marked for translation
- [ ] RTL languages considered
- [ ] No cultural assumptions

## 💬 Helpful Neighbor (Commandment #1)
- [ ] Explains WHY, not just WHAT
- [ ] Shows impact (not just "fix this")
- [ ] Links to learning resources
- [ ] Tone is friendly, not sales-y

## 📚 Drive to Knowledge Base (Commandment #5)
- [ ] KB article exists and linked
- [ ] Article explains problem in plain English
- [ ] Article shows how to fix it yourself (free)
- [ ] Article links to related training

## 🔒 Privacy & Consent (Commandment #10)
- [ ] No data collection without consent
- [ ] Explains what data collected and why
- [ ] User can opt out
- [ ] Privacy policy updated if needed

## 📊 Has a KPI (Commandment #9)
- [ ] Tracks value delivered (time saved, issues fixed, etc)
- [ ] Shows user their progress
- [ ] Demonstrates impact clearly
