# Support This Is My URL Shadow

If This Is My URL Shadow has helped you, there are several meaningful ways to support the project.

This document exists to keep that support request aligned with the project's core philosophy:

- **advice, not pressure**
- **free as possible**
- **community before conversion**
- **clear value before any ask**

---

## The Most Helpful Ways to Support the Project

### 1. Use the Plugin and Share Feedback
Real-world usage helps more than almost anything else.

Please tell us:
- what helped
- what confused you
- what felt trustworthy
- what still needs work

### 2. Leave a Review on WordPress.org
Honest reviews help other site owners discover the project and decide whether it is worth trying.

### 3. Star and Share the Repository
A GitHub star, link, or mention helps improve discoverability and community reach.

### 4. Contribute Documentation, Bug Reports, or Fixes
Clear docs and thoughtful bug reports are some of the most valuable contributions a project like this can receive.

See: [`CONTRIBUTING.md`](CONTRIBUTING.md)

### 5. Sponsor the Work
If GitHub Sponsors is enabled for the project, the **Sponsor** button on GitHub is the preferred way to contribute financially.

If additional support options are introduced later, they should always be explained transparently and without pressure.

---

## What Support Helps Fund

Support makes it easier to invest time in:

- accessibility and usability improvements
- bug fixing and maintenance
- clearer documentation and knowledge base content
- safer diagnostics and treatments
- community support and onboarding
- release quality and testing

---

## Our Promise

Financial support should never turn This Is My URL Shadow into nagware.

We do **not** want support asks to feel like:
- guilt
- fear
- fake urgency
- pay-to-understand-your-own-site behavior

If the project asks for support, it should be because the work has already been helpful and you freely want to keep it going.

---

## Thank You

Whether you support This Is My URL Shadow with time, feedback, reviews, code, or money: thank you.

That support helps keep the project useful, humane, and sustainable.
