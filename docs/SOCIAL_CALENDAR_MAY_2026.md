## Overview

This is the canonical 30-day social content plan for WPShadow Core's May 2026 publishing cycle. All content follows the Helpful Neighbor identity, the 12 Commandments, and the 5 CANON Pillars.

### Content mix

| Type | Target | Posts |
|---|---|---|
| Educational | 60% | ~18 posts |
| Entertaining / relatable | 25% | ~7 posts |
| Product | 15% | 3 posts (May 1, 14, 26) |

### Channels

| Channel | Role |
|---|---|
| X / Twitter | Discovery + threads (bite-sized teaching) |
| LinkedIn | Credibility + relationship-building (process, philosophy) |
| Facebook | Approachable education + weekly routines |
| Instagram | Save-worthy carousels + short screen-capture Reels |
| YouTube | Evergreen how-tos + Shorts |
| Reddit | Trust-building via genuine help (checklists, calm troubleshooting) |

### Content pillars

| Pillar | Description |
|---|---|
| 📅 Calendar dates | Observances and site ops hooks (GAAD, World Password Day, etc.) |
| ⚡ 1-minute win | Tiny improvements any site owner can make right now |
| 💡 Explain it | Plain-language WordPress concepts |
| 🛡️ Fix it safely | Backup/rollback mindset + calm recovery paths |
| 🤝 People & community | Team spotlights, contributor gratitude, open source appreciation |
| 🚀 Product | What WPShadow Core does — what changed, why it matters, what to do next |

### Naming standards (enforced throughout)

- **WPShadow Core** = the free plugin
- **WPShadow Guardian** = the local monitoring feature *inside* WPShadow Core (not a separate plugin)
- **WPShadow Cloud** = planned SaaS platform (reference as hypothetical only)
- Never: "Guardian" alone, "Install the WPShadow Guardian plugin", or any implication of a paid requirement

## Tone and naming reminders

All posts must pass the **Helpful Neighbor test:** "Would a genuinely helpful neighbor say this?" If not, rewrite before publishing.

| Do | Avoid |
|---|---|
| "Let's improve your site's speed" | "Your site is at risk" |
| "Here's one small step" | "You must act now" |
| "WPShadow Core can help you spot this" | "Upgrade to fix" |
| "WPShadow Core" or "WPShadow Guardian (the local monitoring feature inside WPShadow Core)" | "Guardian" alone / "the Guardian plugin" |
| "WPShadow Cloud is planned for the future" | Any implication Cloud is currently available |
