## Revenue Path Framework

### Fix Path Classifications

Every diagnostic falls into one of these fix paths:

| Path | Description | Philosophy Alignment | Examples |
|------|-------------|---------------------|----------|
| **FREE-FIX** | Auto-fixable locally, free forever | Commandments #2, #7 | SSL enforce, file editors disable, permalinks, emoji cleanup |
| **PRO-CORE** | Complex/batch local fixes, requires Pro | Commandment #3 (register-not-pay) | Safe updates, DB optimization, batch accessibility fixes |
| **PRO-MODULE** | Specialized domain fixes, à la carte | Commandment #7 (ridiculously good) | Commerce CRO, media pipelines, compliance templates |
| **SAAS-AI** | Requires external data or AI reasoning | Commandment #2 (free as possible) + #3 | Competitive benchmarking, policy generation, external probes |
| **INSIGHT-ONLY** | No auto-fix, education + manual guidance | Commandments #5, #6 (drive to KB/training) | Business metrics, engagement analytics |

### Monetization Philosophy

**Core Principle:** Diagnostics are always free. We monetize the *fix* or *intelligence*.

- **Free Plugin:** Runs all diagnostics + generous auto-fix set (impressive value)
- **Pro Core:** Advanced local fixes + automation + white-label ($99-199/year/site or $299-499/year/unlimited sites)
- **Pro Modules:** Specialized domain expertise ($49-99/year each or bundled)
- **SaaS/AI:** External intelligence + AI assistance (generous free tier via registration → usage-based upsell)
