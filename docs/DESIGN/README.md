## 🌟 Core Values

Design reflects our commitment to:
- ✅ **Pillar 1:** 🌍 Accessibility First (WCAG 2.1 AA)
- ✅ **Pillar 2:** 🎓 Learning Inclusive (clear for all)
- ✅ **Pillar 3:** 🌐 Culturally Respectful (inclusive design)
- ✅ **Commandment #1:** Helpful Neighbor Experience (intuitive UI)

Learn more: [PHILOSOPHY/VISION.md](../PHILOSOPHY/VISION.md)

## ♿ Accessibility First

All components are:
✅ WCAG 2.1 AA compliant  
✅ Keyboard navigable  
✅ Screen reader compatible  
✅ Color contrast validated  
✅ Focus indicators visible  

Details: [GUIDELINES.md](GUIDELINES.md)

## 🎨 Design Principles

1. **Accessibility First** - No features without accessibility
2. **Learning Inclusive** - Visual language clear to everyone
3. **Culturally Respectful** - No cultural assumptions
4. **Helpful Neighbor** - Intuitive, not surprising
5. **Responsive** - Works on all screen sizes

See: [GUIDELINES.md](GUIDELINES.md)
