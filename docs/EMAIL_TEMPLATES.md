## Usage Guidelines

### When to Use Each Template

| Template | Timing | Trigger | Frequency |
|----------|--------|---------|-----------|
| **Welcome** | Day 3 | After opt-in | Once |
| **Quick Win** | Day 7 | First week complete | Once |
| **Educational** | Day 14 | Onboarding | Once, then quarterly |
| **Monthly Summary** | Monthly | First of month | Monthly |
| **Re-Engagement** | Variable | Engagement score < 25 | Once, then pause |
| **Milestone** | Variable | Achievement reached | As earned |
| **Feature Announcement** | Variable | New feature matches usage | Ad-hoc |
| **Content/Education** | Variable | Manual trigger | 1-2x per quarter |

### Personalization Best Practices

1. **Always use real data** - Never fake or approximate numbers
2. **Test variables** - Ensure all {{variables}} have fallbacks
3. **Segment appropriately** - Don't send workflow tips to non-workflow users
4. **Respect engagement** - Check engagement score before every send
5. **A/B test** - Try different subject lines and measure opens

### Writing Tips

- ✅ **Be conversational** - Write like you're emailing a friend
- ✅ **Be specific** - "47 issues fixed" not "many issues fixed"
- ✅ **Be helpful** - Education over promotion
- ✅ **Be brief** - Aim for 2-minute read time
- ✅ **Be actionable** - Always include next steps
- ❌ **Avoid jargon** - Plain English only
- ❌ **Avoid scare tactics** - Confidence, not fear
- ❌ **Avoid pressure** - No urgency or FOMO tactics
