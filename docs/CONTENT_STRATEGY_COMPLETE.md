## 🎓 Lessons Learned

### What Worked Well
1. **Bash/curl/jq approach** more reliable than Python in Codespaces
2. **Batch size of 5-10 issues** optimal for rate limiting
3. **Compact script format** improved debugging and visibility
4. **2.5 second sleep intervals** perfectly balanced speed and reliability
5. **Comprehensive documentation first** made automation easier

### Challenges Overcome
1. **Python requests module unavailable** → Switched to bash/curl
2. **GitHub rate limiting** → Added sleep intervals and retry logic
3. **Large batch failures** → Reduced batch size from 20 to 5-10
4. **Duplicate issues created** → Improved tracking and verification
5. **Script interruptions** → Added resume capability
