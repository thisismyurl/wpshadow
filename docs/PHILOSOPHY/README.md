## 🌟 The Philosophy

### The 11 Commandments
1. **Helpful Neighbor Experience** - Education over selling
2. **Free as Possible** - No artificial limitations
3. **Register, Don't Pay** - Fair exchange when needed
4. **Advice, Not Sales** - Educational approach
5. **Drive to Knowledge Base** - Link to learning
6. **Drive to Free Training** - Education first
7. **Ridiculously Good for Free** - Quality bar
8. **Inspire Confidence** - Empower users
9. **Everything Has a KPI** - Measure impact
10. **Beyond Pure** - Privacy first
11. **Talk-About-Worthy** - Share-worthy features

**Full details:** [VISION.md](VISION.md)

### The 3 Accessibility Pillars
1. **🌍 Accessibility First** - No compromises
2. **🎓 Learning Inclusive** - Multiple modalities
3. **🌐 Culturally Respectful** - Global community

**Full details:** [ACCESSIBILITY.md](ACCESSIBILITY.md)

## 📖 Key Concepts

### Helpful Neighbor, Not Sales Tool
WPShadow exists to help WordPress users succeed, not to sell them upgrades.
- We educate first, promote learning
- We keep features free when possible
- We're transparent about when costs are needed
- We measure impact with KPIs

### Education Over Everything
Teaching is core to our identity:
- Every feature includes learning links
- Tooltips explain the "why" not just the "what"
- Documentation is comprehensive and clear
- Free training is always available

### Accessibility is Non-Negotiable
We design for everyone:
- WCAG 2.1 AA compliance throughout
- Works with assistive technologies
- Multiple learning formats supported
- Respectful of cultural differences
