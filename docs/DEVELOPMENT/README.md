## 🌟 Core Values

Development is guided by:
- ✅ **Commandment #3:** Register, Don't Pay (open source, no barriers)
- ✅ **Commandment #6:** Drive to Free Training (resources available)
- ✅ **Commandment #10:** Beyond Pure (security practices)
- ✅ **Pillar 2:** 🎓 Learning Inclusive (code examples for all)

Learn more: [PHILOSOPHY/VISION.md](../PHILOSOPHY/VISION.md)
