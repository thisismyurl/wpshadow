## 🚀 Quick Start (3 Steps)

### 1️⃣ Download
Download `wpshadow-1.2601.2148.zip` from this repository.

### 2️⃣ Install
**Option A - WordPress Admin (Easiest):**
```
WordPress Admin → Plugins → Add New → Upload Plugin
Select wpshadow-1.2601.2148.zip → Install Now → Activate
```

**Option B - Manual Upload:**
```
Extract ZIP → Upload "wpshadow" folder to wp-content/plugins/ → Activate
```

**Option C - Command Line:**
```bash
wp plugin install wpshadow-1.2601.2148.zip --activate
```

### 3️⃣ Test
```
1. Go to WPShadow menu in WordPress admin
2. Run initial diagnostics
3. Explore the dashboard
4. Test features and report any issues
```
